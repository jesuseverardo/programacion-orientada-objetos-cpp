# Prácticas de Programación Orientada a Objetos en C++

Este repositorio contiene una colección de prácticas y ejercicios desarrollados en C++ aplicando conceptos de Programación Orientada a Objetos.

Los programas abordan temas como clases, objetos, constructores, herencia, apuntadores, colecciones, vectores y sobrecarga de operadores. Cada práctica está organizada por parcial y puede compilarse de forma independiente.

## Temas incluidos

- Clases y objetos
- Constructores
- Métodos y atributos
- Encapsulamiento
- Herencia
- Apuntadores
- Vectores y colecciones
- Sobrecarga de operadores
- Entrada y salida de datos
- Compilación independiente de archivos `.cpp`

## Requisitos

Para compilar y ejecutar los programas se necesita:

- Un compilador compatible con C++17.
- Visual Studio 2019 o posterior, GCC o Clang.

## Objetivo

El objetivo de este repositorio es documentar prácticas académicas de Programación Orientada a Objetos en C++, mostrando ejemplos básicos y progresivos que ayuden a reforzar los conceptos vistos en clase.
